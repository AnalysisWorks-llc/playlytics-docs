# Analytics Overview

Playlytics turns raw game and practice data into visual analytics that coaches can act on. Analytics are available for all four areas of the game plus mechanics assessments.

## Accessing Analytics

Navigate to **Analytics** from the bottom navigation or More menu. Select a player from the roster to view individual analytics, or view Team Analytics for aggregate stats.

## Analytics Types

**Pitching Analytics** — heat maps, velocity trends, pitch type distribution, strikeout and walk rates, ERA

**Hitting Analytics** — spray chart, batting average trend, contact type distribution, hard hit rate, OBP and slugging

**Fielding Analytics** — fielding percentage, zone tendency polar charts, throwing accuracy, error analysis

**Baserunning Analytics** — stolen base percentage, caught stealing rate, per-base breakdown

**Mechanics Assessments** — radar charts for all four mechanics areas with phase breakdowns and trend lines

## Event Type Filter

All analytics screens include an event type filter:
- **All** — includes game, scrimmage, team practice, and individual practice data
- **Game** — game data only
- **Scrimmage** — scrimmage data only
- **Team Practice** — practice data only
- **Individual Practice** — individual session data only

Use this filter to compare in-game performance against practice performance.

## Date Range Filter

Filter analytics to a specific date range — useful for evaluating performance over a particular stretch of games or comparing before and after a training intervention.

## Minimum Sample Sizes

Analytics become meaningful with sufficient data. As a general guide:
- Batting average: meaningful after 20+ at-bats
- ERA: meaningful after 10+ innings pitched
- Fielding percentage: meaningful after 20+ fielding chances
- Mechanics assessments: meaningful after 5+ sessions
