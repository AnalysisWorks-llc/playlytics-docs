# Player Report Cards

## Overview

Player report cards compile a player's stats, analytics charts, mechanics assessments, and coach comments into a shareable document. They are designed to communicate player development progress to families in a professional format.

## Generating a Report Card

1. Navigate to a player's profile
2. Tap **Generate Report Card**
3. Select the date range — season to date or a specific period
4. Add coach comments (optional but recommended)
5. The report card generates automatically

## Report Card Contents

- **Player header** — name, jersey number, team, season
- **Season stats** — key counting stats for all four areas
- **Performance charts** — trend charts for key metrics
- **Mechanics assessment** — radar charts for all four mechanics areas with top strength and top focus area
- **Training plan summary** — active training plans
- **Coach comments** — free text from the coach

## Sharing with Families

Tap **Share** to generate a shareable link. The link:
- Works for anyone — no Playlytics account required
- Expires after 30 days
- Shows a read-only version of the report card
- Can be shared via text, email, or any messaging app

## PDF Export

Tap **Export PDF** to generate a PDF version of the report card suitable for printing or attaching to an email.

## Coach Comments Best Practices

The mechanics assessment section provides specific data to reference in your comments:
- Mention specific strengths by name ("Your hip rotation score of 2.8 shows consistent technique")
- Reference focus areas with context ("We are working on your arm path — the trend is improving")
- Connect observations to training plans ("Continue the changeup command drills we started in March")

Specific, data-referenced comments are more valuable to families than general feedback.
