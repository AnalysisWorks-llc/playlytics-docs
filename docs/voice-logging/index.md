# Voice Logging

Voice logging lets coaches and parent volunteers describe plays naturally using their voice. The AI interprets the description and logs it to the correct fields automatically.

## How It Works

1. Tap **Voice Log** from the Game Hub or Practice Hub
2. Describe the play as you would tell someone what happened
3. Review the AI-parsed result on the confirmation card
4. Confirm to save or edit if needed

The AI handles all four logger types — pitching, hitting, fielding, and baserunning — and determines which type based on the description.

## Why Voice Logging

Voice logging solves the volunteer logger problem. A parent who has never used Playlytics can log data accurately by simply describing what they see. No need to learn the app layout or navigate between screens.

Voice logging is especially valuable at practices where repetitions happen faster than manual entry can keep up.
