# Mobile Voice Logging

Mobile voice logging uses the device keyboard's microphone input. On Android with Google Voice Typing this gives unlimited recording time with no auto-timeout.

## Android Setup (One Time)

1. Go to **Settings → General Management → Keyboard list and default**
2. Set **Google Voice Typing** as your default input method

After this setup the microphone icon appears on your keyboard whenever a text field is active.

## How to Use on Android

1. Open Voice Logger from the Game Hub
2. The text field activates and your keyboard appears
3. Tap the **microphone icon** on the keyboard
4. Speak your play description — you can speak as long as needed
5. Tap the **arrow/send button** when done speaking
6. The AI processes your description and shows the confirmation card

**Key difference from desktop:** The keyboard microphone does not auto-stop. Speak the complete play, then tap the send button manually. This gives you full control and avoids timeout issues.

## iOS Setup

No setup required. Safari handles voice input natively.

## How to Use on iOS

1. Open Voice Logger from the Game Hub
2. Tap the **microphone icon** in the voice logging interface
3. Speak your play description
4. The recording stops automatically
5. Review and confirm on the confirmation card

## Troubleshooting Mobile

**Microphone icon not showing on Android keyboard**
Confirm Google Voice Typing is set as your default keyboard. Some Android manufacturers use a custom keyboard as default — Google Voice Typing must be set explicitly.

**App asks for microphone permission**
Tap Allow. If you denied it previously go to **Settings → Apps → Playlytics → Permissions → Microphone → Allow**.

**Safari blocks microphone on iOS**
Go to **Settings → Safari → Microphone** and ensure it is set to Ask or Allow.
