# Managing Your Roster

## Adding Players

Add players at any time — you do not need a complete roster before logging game data.

1. Navigate to **Roster**
2. Tap **+ Add Player**
3. Enter first name, last name, and jersey number
4. Select positions — mark primary and secondary positions
5. Tap **Save**

Players appear on the roster immediately and are available in all loggers.

## Positions

Select all positions a player can play. The defensive lineup builder uses position data to suggest alignments — a player only appears as a candidate for positions where they have been marked.

Standard positions: P, C, 1B, 2B, 3B, SS, LF, CF, RF, DH, UT

## Jersey Numbers

Jersey numbers are used to distinguish players in analytics displays and on report cards. Numbers do not need to be unique — if two players share a number both names appear and are distinguishable by their full name.

## Editing and Correcting Records

Open a player profile and tap **Edit** to change any field. Changes apply going forward — historical game data is not retroactively altered when you edit player information.

## Archiving vs Deleting

**Archive** — use when a player leaves the team mid-season. All historical data is preserved. The player is hidden from active roster views and loggers but their stats remain in team analytics for the period they were active.

**Delete** — permanently removes the player and all associated data. This cannot be undone. Only delete players who were added by mistake and have no real game data attached.

## Re-ordering the Roster

The roster displays alphabetically by last name by default. Drag players to reorder if you prefer a different arrangement — batting order, position group, or jersey number order.

## Bulk Import

Contact support at support@playlytics.app if you need to bulk-import a roster from a spreadsheet. This is available for Organization plan subscribers setting up multiple teams.
