# Practice Sessions

Playlytics supports two practice session types: Team Practice and Individual Practice. Practice data flows into the same analytics as game data and can be filtered separately using the event type filter.

## Session Types

**Team Practice** — whole-team practice. Use this for bullpen sessions, batting practice, fielding drills, and intra-squad work. Multiple players are tracked in the same session.

**Individual Practice** — focused work with one player. Used for mechanics assessments, targeted skill development, and one-on-one coaching sessions. This is where mechanics assessments are conducted.

## Starting a Practice Session

1. From the dashboard tap **Log Practice Data**
2. Select **Team Practice** or **Individual Practice**
3. For Individual Practice select the player
4. Tap **Start Session**

The Practice Hub opens with the same four loggers available as in Game mode.

## Key Differences from Game Mode

- No opponent or score tracker
- No inning tracker
- No game result to record at the end
- Individual Practice includes the Mechanics Assessment logger

## Ending a Practice Session

Tap **End Session** from the Practice Hub. You are not required to record a result — just confirm you want to end the session.

Practice sessions appear in the session history with the date and duration. Data from all practice sessions is included in analytics (with the event type filter set to All or Team/Individual Practice).
