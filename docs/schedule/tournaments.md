# Tournament Management

Playlytics handles tournament scheduling with support for pool play, bracket games, TBD opponents, and final placement recording.

## Adding a Tournament

1. Navigate to **Schedule**
2. Tap **+ Tournament**
3. Enter:
   - **Tournament name** — e.g. "Midwest Invitational"
   - **Location** — city or facility name
   - **Start date** and **End date**
   - **Format** — Pool Play + Bracket, Round Robin, or Single Elimination
4. Tap **Save**

The tournament appears as a grouped card on your schedule showing all games within it.

## Adding Pool Play Games

From the tournament detail screen tap **+ Game** to add pool play games. Pool play opponents are usually known in advance — select or add the opponent for each game slot.

## Adding Bracket Games

Bracket games are added the same way. For games where the opponent is not yet known:

1. Tap **+ Game**
2. Select **TBD** as the opponent
3. Set the date and approximate time
4. Save

When the bracket is released edit each game to replace TBD with the actual opponent.

## Scoring Tournament Games

Score each tournament game exactly as you would a regular season game. Start a game session, select the tournament game, and score normally. The game links automatically to the tournament.

## Recording Tournament Placement

When the tournament is over:

1. Open the tournament detail screen
2. Tap **Mark Complete**
3. Select your final placement:
   - Champion
   - Runner-Up
   - Semi-Final
   - Quarter-Final
   - Pool Play Only (did not advance to bracket)
4. Tap **Save**

Champion and Runner-Up placements display as trophy badges on the tournament card. Both badges appear in your season summary on the dashboard.
